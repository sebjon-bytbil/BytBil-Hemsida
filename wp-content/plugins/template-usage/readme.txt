Template Usage gives you a simple overview page of all page templates in your site, allowing you to easily find pages using a specific template.

== Description ==

This plugin gives you an overview of page templates in your site. You get both an overview of all templates and how many pages they have. You also get a detailed view of all the pages using a specific template.

Usage for this can be to identify unused templates and to find pages using specific templates to change them in some way.

== Installation ==

1. Upload Template Usage to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the Settings > Template Usage page in the admin gui.
